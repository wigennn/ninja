领域模型设计

领域模型设计

本文档介绍如何设计领域模型。

识别实体和值对象

实体特征

  • 有唯一标识
  • 生命周期可追踪
  • 可变

值对象特征

  • 无唯一标识
  • 不可变
  • 通过值相等性比较

设计聚合

聚合应该：

  • 保持小
  • 通过ID引用其他聚合
  • 维护不变性约束

示例

用户聚合：

  • User（聚合根）
  • UserStatus（值对象）
  • UserRepository（仓储接口）
  • UserDomainService（领域服务）
